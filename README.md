# opendatainitiative.io

[Jekyll](https://jekyllrb.com/docs/home/) source code for The Open Data Initiative website.

## Requirements
- Ruby v2.3.3
- [Bundler](http://bundler.io/)

## Installation
```
1. git clone <repo>
2. cd opendatainitiative.io
3. bundle install
```

## Serving
```
bundle exec jekyll serve
```
